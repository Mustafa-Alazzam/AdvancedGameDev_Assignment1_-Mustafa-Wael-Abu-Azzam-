# Advanced Game Development - Assignment 1
## Dragon Jump Prototype

**Student Name:** Mustafa Wael Abu Azzam  
**Unity Version:** 2021.3.45f1 LTS

### Project Description
A 2D side-scrolling prototype where the player controls a Dragon. The goal is to jump over obstacles to survive. This project demonstrates basic Unity physics, player input, and version control using Git.

### Implemented Features
* **Player Controller:** Smooth movement and jumping mechanics.
* **Physics:** Used Rigidbody2D and BoxCollider2D for realistic interactions.
* **Flip Logic:** The Dragon changes direction based on movement input.
* **Obstacles:** Environment setup with collision detection.

### How to Play
* **Space Bar / Up Arrow:** Jump.
* **A/D or Left/Right Arrows:** Move the Dragon.
